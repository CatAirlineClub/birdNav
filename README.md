# birdNav - 飞鸟导航
vercel地址: https://bird-nav.vercel.app  
一颗追求极致简约、极致实用的心，做一个功能强大而简洁的网址导航  

### 参与共建方式
为了减少代码冲突，尽量是fork然后提交pr，由审核员测试再合并。  
如果不会的话可以在分支开发完merge进主分支  
尽量不要在主分支直接改然后push哈~
